Transformée de Fourier — 1D

import numpy as np, matplotlib.pyplot as plt
N, Fs = 16, 2.0; Ts = 1/Fs
t = np.arange(0, N, Ts)
sig = np.sin(2*np.pi*t*4/16)                 # 4 périodes sur 16
F = np.fft.fft(sig); freq = np.fft.fftfreq(F.size, d=Ts)
fig, ax = plt.subplots(1, 2, figsize=(7, 2.7))
ax[0].plot(t, sig, 'r.-'); ax[0].set_title("Signal périodique")
ax[1].plot(freq, np.abs(F), 'r.-'); ax[1].set_title("Composantes de Fourier")
plt.show()

Filtrage fréquentiel

  • Passe-bas : ne garde que les basses fréquences → lissage
  • Passe-haut : ne garde que les hautes fréquences → détails/contours
  • Passe-bande : une bande de fréquences précise
  • Filtrer = multiplier le spectre par la réponse du filtre

L’aliasing

Sous-échantillonner sans filtrer → repliement de spectre :

import rioxarray as rxr
from scipy.ndimage import gaussian_filter
img = rxr.open_rasterio('../berkeley.jpg').to_numpy()       # (3, H, W)
fig, ax = plt.subplots(1, 3, figsize=(8, 3))
ax[0].imshow(img.transpose(1,2,0).astype('uint8')); ax[0].set_title("Originale"); ax[0].axis('off')
ax[1].imshow(img[:, ::4, ::4].transpose(1,2,0).astype('uint8')); ax[1].set_title("Décimée x4"); ax[1].axis('off')
lisse = gaussian_filter(img.astype(float), sigma=(0, 1.5, 1.5))
ax[2].imshow(lisse[:, ::4, ::4].transpose(1,2,0).astype('uint8')); ax[2].set_title("Passe-bas puis décimée"); ax[2].axis('off')
plt.show()

Filtrage d’image

  • Objectifs : réduire le bruit, réhausser, extraire des caractéristiques
  • Global (Fourier) vs local (fenêtre glissante)
  • Linéaire (moyenne, convolution) vs non linéaire
  • Stationnaire vs adaptatif

Filtre moyen & contours (Scharr)

from scipy import signal
scharr = np.array([[-3-3j, 0-10j, 3-3j], [-10+0j, 0+0j, 10+0j], [-3+3j, 0+10j, 3+3j]])
gris = img[0]                                    # une bande
grad = np.abs(signal.convolve2d(gris, scharr, boundary='symm', mode='same'))
fig, ax = plt.subplots(1, 2, figsize=(7, 3))
ax[0].imshow(gris, cmap='gray'); ax[0].set_title("Originale"); ax[0].axis('off')
ax[1].imshow(grad, cmap='gray'); ax[1].set_title("Contours — filtre de Scharr"); ax[1].axis('off')
plt.show()

Filtrage adaptatif — le speckle SAR

Les images radar sont dégradées par un bruit multiplicatif (speckle). Le filtre de Lee lisse en préservant les détails (pondération selon la variance locale).

sar = rxr.open_rasterio('../SAR.tif').sel(band=1)
sar = sar.where(sar > 0)                          # masque no_data
moy = sar.rolling(x=5, y=5, min_periods=3, center=True).mean()
fig, ax = plt.subplots(1, 2, figsize=(7, 3))
np.log10(sar).plot.imshow(ax=ax[0], add_colorbar=False); ax[0].set_title("SAR brut (speckle)"); ax[0].axis('off')
np.log10(moy).plot.imshow(ax=ax[1], add_colorbar=False); ax[1].set_title("Filtre moyen 5x5"); ax[1].axis('off')
plt.show()

Segmentation en super-pixels

SLIC regroupe les pixels en régions homogènes et compactes (couleur + distance) :

from skimage.segmentation import slic, mark_boundaries
rgb = img.transpose(1, 2, 0).astype('uint8')      # berkeley (réutilisée)
seg = slic(rgb, n_segments=250, compactness=10, sigma=1, start_label=1)
fig, ax = plt.subplots(figsize=(5, 3.6))
ax.imshow(mark_boundaries(rgb, seg))
ax.set_title(f"SLIC — {len(np.unique(seg))} super-pixels"); ax.axis('off')
plt.show()

Fusion et approche objet

  • Sur-segmentation → fusionner les segments similaires (graphe d’adjacence, RAG)
  • Approche objet : chaque segment devient un objet décrit par des propriétés (aire, moyenne, forme…) via regionprops
  • Base de la classification objet (chapitre suivant)

Points clés

  • La transformée de Fourier : point de vue fréquentiel réversible
  • Passe-bas avant de décimer → éviter l’aliasing
  • Convolution = filtrage local efficace (moyenne, Scharr)
  • Lee : filtre adaptatif contre le speckle SAR
  • SLIC + RAG + regionprops : de l’image aux objets
